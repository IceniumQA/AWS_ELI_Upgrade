Amazon Web Services Plugin Demo

The Amazon Web Services (AWS) plugin wraps the Javascript SDK so you can easily access a host of cloud-based services. This demo showcases only a few of the possibilities.

Important
Make sure you specify the ACCESS_KEY and SECRET_KEY parameters in the plugin settings.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.
